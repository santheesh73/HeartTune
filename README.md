## 🌟 Overview

**HeartTune** is a modern full-stack music streaming platform inspired by Spotify and Apple Music.

Built using **Next.js**, **TypeScript**, **Supabase**, and the **JioSaavn API**, HeartTune delivers a beautiful and responsive listening experience with authentication, playlists, likes, downloads, recently played history, and Progressive Web App support.

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/hearttune.git

cd hearttune
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:5175
```
