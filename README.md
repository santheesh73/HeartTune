# HeartWave

A Spotify-style music streaming app with a red and black theme, powered by the [JioSaavn API](https://saavn.sumit.co).

## Features

- Login and registration
- Music search (songs, albums, playlists) via JioSaavn
- Full music player with shuffle, repeat, and volume control
- Liked songs library
- Offline downloads stored in the browser
- Album browsing and playback
- Red and black UI with animations

## Getting Started

```bash
cd heartwave
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Usage

1. Sign up with any email and password (minimum 6 characters)
2. Search for songs on the Search page
3. Play music by clicking any song
4. Like songs with the heart icon
5. Download songs for offline listening
